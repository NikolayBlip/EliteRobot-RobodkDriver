# Avatar

Проект для передачи в реальном времени позы с аватара кобота на EliteRobot кобота через servoj.

## 📋 Описание

Проект реализует управление EliteRobot коботом через аватар в реальном времени. Углы сочленений передаются для синхронного воспроизведения движений.

### Вариант 1: C++ монолитная прошивка
- Единый C++ скрипт для прошивки аватара
- Встроенные функции:
  - RTSI коммуникация
  - Отладочные данные
  - Калибровка углов
  - Перевод углов в радианы

### Вариант 2: Python/C++ гибрид
- C++ компонент на аватаре для передачи данных через serial
- Python скрипт на компьютере для:
  - Приема данных через serial
  - Отправки через RTSI на реального кобота

## 🔧 Установка

1. Выберите нужный вариант реализации
2. Загрузите соответствующую папку проекта

## 🔧 Использование
При "монолитной" прошивке:
  0. Загрузите папку RTSI Frimware
  1. Укажите правильные сетевые настройки перед прошивкой teensy:
  ```cpp
    #define STATIC_IP     IPAddress(192, 168, 0, 0); // <<< СВОБОДНЫЙ IP-адрес для Teensy
    #define SUBNET_MASK   IPAddress(255, 255, 255, 0);  // <<< маска подсети (обычно 255.255.255.0)
    #define GATEWAY_IP    IPAddress(192, 168, 0, 1);   // <<< IP-адрес вашего роутера/шлюза
    #define DNS_IP        IPAddress(8, 8, 8, 8);       // <<< IP-адрес DNS-сервера (необязательно для локальной сети)
    #define ROBOT_IP IPAddress(192, 168, 0, 10); // <<< IP-АДРЕС ВАШЕГО РОБОТА
   ```
  2. Подключите аватар к Elite коботу
  3. Запустите выбранную реализацию
  4. Начните передачу углов сочленений в реальном времени
